# 🔐 IoT SecureNet Platform v3.0

Plateforme de supervision et sécurité pour réseaux IoT, avec Digital Twin interactif.

## Fonctionnalités
- 🌐 **Digital Twin** — topologie réseau interactive
- ⚔️ **Simulation d'attaques** — 12 types (SYN Flood, MITM, Firmware Exploit...)
- 🛠️ **Actions réseau** — 14 actions (bloquer, patch, quarantaine, VLAN...)
- 📡 **Scanner réseau** — auto-découverte IP/MAC/vendeur
- 💻 **Infos système** — CPU/RAM/ROM en temps réel
- 🎬 **SplashScreen animé** — démarrage professionnel

## Stack technique
- Java 17+ · Swing · FlatLaf Dark · SQLite / MySQL
- Maven · EventBus · SwingWorker

## Démarrage rapide
```bash
mvn clean package
java -jar target/iot-securenet-*.jar
```
